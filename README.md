# Budget Tracker

A simple browser budget tracker for weekly spending, category breakdowns, and CSV statement imports.

## Features

- Set a weekly budget and track weekly/monthly spend
- Add transactions manually or import bank/credit-card CSV statements
- Auto-categorize spend (groceries, dining, transport, and more)
- Search and recategorize transactions
- Data saved in your browser via `localStorage`

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080

## Privacy

All data stays in your browser. Nothing is sent to a server.
